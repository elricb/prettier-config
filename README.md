# prettier-config

Personal [Prettier](https://prettier.io) config for my projects.

## Install

```
$ npm install --save-dev @elricb/prettier-config
```

## Usage

Add this to your `package.json`:

```json
{
	"prettier": "@elricb/prettier-config"
}
```
